# 動漫資訊站專案規劃

## 作品定位
這是一個日本動漫資訊整理網站，不提供盜版影片或侵權內容。

主要提供：
- 動畫資料介紹
- 串流平台資訊整理
- 搜尋與分類功能
- 動畫詳細頁

### 1. 首頁
- 網站標題與介紹
- 精選動畫區
- 熱門分類入口
- 平台入口

### 2. 動畫列表頁
- 顯示全部動畫卡片
- 關鍵字搜尋
- 類型篩選
- 平台篩選

### 3. 動畫詳細頁
- 動畫名稱
- 海報圖片
- 動畫介紹
- 類型
- 集數
- 播出季度
- 可觀看平台

### 4. 串流平台頁
- 平台名稱
- 平台介紹
- 平台作品數量
- 平台收錄動畫

### 5. 管理頁
- 新增動畫
- 修改動畫
- 刪除動畫

## 開發順序
### 第一階段：規劃與假資料
- 確定欄位
- 畫面骨架
- 假資料卡片頁

### 第二階段：後端 API
- 建立 Express
- 建立動畫 API
- 建立平台 API

### 第三階段：前後端串接
- 前端改用 fetch 拿資料
- 詳細頁接單筆 API

### 第四階段：CRUD
- 新增動畫
- 修改動畫
- 刪除動畫

### 第五階段：資料庫
- 將假資料改成 SQLite
